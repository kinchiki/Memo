## ショートカット
| キー | 動作 |
|:--|:--|
| cmd+クリック | メニュー表示 |
| ctrl+cmd+←→ | 戻る、進む |
| ctrl+cmd+J | 定義元にジャンプ |
| shift+cmd+J | 開いているファイルに飛ぶ |
| cmd+1,2,3,... | 左サイドバー |
| shift+cmd+K | Clean |
| cmd+. | ビルドキャンセル |
| cmd+option+Enter or option+click | ウィンドウ分割 |
| cmd+Enter | ウィンドウ分割終了 |
| ctrl+cmd+r | Runオンリー |


## Tips
- ブレークポイント削除 枠外にドラッグ
- Editor > Show Invisibles
